# 📚 Guia GitHub para Iniciantes

## 📦 Conteúdo da Pasta

Esta pasta contém um guia completo e didático para iniciantes aprenderam a usar GitHub e Git no Windows.

### Arquivos Disponíveis

1. **`guia_github_iniciantes.md`** (18 KB)
   - Versão em Markdown do guia
   - Ideal para ler no GitHub ou em editores de texto
   - Fácil de modificar e personalizar

2. **`guia_github_iniciantes.pdf`** (176 KB)
   - Versão em PDF formatada e pronta para imprimir
   - Com índice automático (Table of Contents)
   - Melhor para ler em tablets ou imprimir em papel

3. **`screenshots/`** (pasta)
   - Armazena capturas de tela dos processos
   - Usadas como referências visuais no guia

4. **`README.md`** (este arquivo)
   - Informações sobre os arquivos

1. **O que é GitHub?** - Introdução ao conceito
2. **Criando uma Conta** - Passo a passo com campos explicados

## 🚀 Como Começar

1. Leia a seção **"O que é GitHub?"** para entender os conceitos
2. Siga passo a passo as instruções de **"Criando uma Conta"**
3. Pratique com o guia **"Enviando Arquivos para o GitHub"**
4. Consulte **"Erros Comuns"** se encontrar problemas
