📖 INSTRUÇÕES DE USO
═══════════════════════════════════════════════════════════════════════════

PRIMEIRA LEITURA:
  1. Abra "index.html" em um navegador
  2. Comece com "Guia Principal" (markdown ou PDF)
  3. Leia a primeira seção: "O que é GitHub?"
  4. Siga passo a passo cada instrução
  5. Execute os comandos conforme aprende

DURANTE A PRÁTICA:
  1. Mantenha o "Guia Principal" aberto
  2. Consulte "Referência Rápida" para comandos
  3. Veja "Diagramas Visuais" se tiver dúvidas
  4. Practique repetidamente
  5. Não pule nenhuma etapa

COMO REFERÊNCIA:
  1. Imprima o PDF se preferir papel
  2. Consulte rápido a "Referência Rápida"
  3. Volte ao tópico específico quando precisar
  4. Use o "Índice" para encontrar rapidamente
