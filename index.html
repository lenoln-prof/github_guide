💡Dicas Importantes

O GitHub não aceita mais sua senha comum. Você PRECISA de um Personal Access Token para fazer push!
Pratique regularmente - é assim que você aprende de verdade. Não tenha medo de cometer erros!

🎓Como Usar Este Guia

  1. Leia na ordem: Comece pelo "Guia Principal" do início ao fim
  2. Pratique: Execute os passos enquanto lê
  3. Consulte a Referência: Use "Referência Rápida" quando precisar dos comandos
  4. Visualize: Veja "Diagramas Visuais" para entender melhor os conceitos
  5. Revise: Volte aos tópicos conforme necessário

🚀Próximos Passos

Após dominar o básico, explore:

  • Branches: Trabalhe em diferentes versões do código
  • Pull Requests: Contribua com projetos de outras pessoas
  • Colaboração: Trabalhe em equipe com amigos e colegas
  • GitHub Pages: Crie um site gratuito com GitHub
  • GitHub Actions: Automatize tarefas (testes, deploy)
